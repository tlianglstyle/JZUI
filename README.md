#JZUI
###使用bootstrap+vuejs库,配合webpack,gulp构建工具搭建的后端js组件库

###install
``` bash
### git
$ git clone https://github.com/tlianglstyle/JZUI.git

### install 
$ npm install  (or 'cnpm install')
### build
$ gulp dev
### open
open  http://localhost:3000/dist/app/Vue/index.html

```
